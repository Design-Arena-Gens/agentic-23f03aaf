## المفتش – لوحة إدارة المفتشية

منصة ويب مبنية على Next.js و Tailwind CSS لتجميع جميع أدوات المفتش في مكان واحد: قاعدة بيانات المديرين والمدارس، إعداد التقارير، جداول الإرسال، متابعة الأداء، جدولة المواعيد، وحجز مراسلات Gmail.

## المزايا الرئيسية

- واجهة دخول آمنة (اسم المستخدم `inspector` وكلمة المرور `admin123`).
- لوحة قيادة شاملة تعرض الإحصائيات والتنبيهات العاجلة.
- إدارة بيانات المديرين والمدارس الابتدائية مع إمكانية الإضافة الفورية.
- أرشفة التقارير الميدانية مع ملخصات زمنية.
- لوحة لمتابعة المديرين وخطط الزيارات.
- جداول خاصة بالمراسلات الرسمية، المهل النهائية، والتذكيرات.
- حاسبة مردودية تقديرية مبنية على نسب النجاح وعدد الأنشطة المنجزة.
- مركز رسائل Gmail لحفظ المسودات وجدولة الإرسال (محاكاة إرسال فوري للاختبار).
- نظام جدولة مواعيد مع تنبيهات للمواعيد القريبة (خلال ٣ أيام).

## التشغيل محلياً

```bash
npm install
npm run dev
```

قم بزيارة http://localhost:3000 ثم استخدم بيانات الدخول أعلاه.

## البناء

```bash
npm run build
```

## النشر على Vercel

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-23f03aaf
```

بعد اكتمال النشر، تحقق من التشغيل:

```bash
curl https://agentic-23f03aaf.vercel.app
```
